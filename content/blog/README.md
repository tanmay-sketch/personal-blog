For math equations, use the following syntax:

The blog post is written in Markdown and can use the following components:
- InlineMath
- BlockMath

Usage of InlineMath:
```
<InlineMath>{"A"}</InlineMath>
```

Usage of BlockMath:
```
<BlockMath>{"A"}</BlockMath>
```
